GitHub Actions 执行流程示意图

数据流说明
抓取数据 → Python Scraper → 输出原始 JSON/CSV
数据清洗 & 分类 → 输出整理后的 JSON
AI 处理 → GPT API 摘要、生成 Markdown
输出 & 存储 → GitHub Repo + 本地存档
通知 → 可选推送到邮箱或 Slack

│        GitHub Actions          │

│ - 手动触发 (workflow_dispatch) │
└───────────────┬───────────────┘
                │
                ▼
┌───────────────────────────────┐
│       Python Scraper           │
│ - 抓取网站/API                 │
│   (STEM4kids, Art School,     │

│ - 输出原始 JSON / CSV 文件      │
└───────────────┬───────────────┘
                │
                ▼
┌───────────────────────────────┐
│       数据清洗 & 预处理        │
│ - 去重、结构化                │
│ - 分类课程类型 (编程/艺术/舞蹈)│
│ - 输出整理好的 JSON 文件       │
└───────────────┬───────────────┘
                │
                ▼

                ▼

└───────────────┬───────────────┘
                │
                ▼
┌───────────────────────────────┐
│         可选通知                │
