# GitHub Actions 执行流程 (可视化版)

## 数据流说明

- **抓取数据** → Python Scraper → 输出原始 JSON/CSV  
- **数据清洗 & 分类** → 输出整理后的 JSON  
- **AI 处理** → GPT API 摘要、生成 Markdown  
- **输出 & 存储** → GitHub Repo + 本地存档  
- **通知** → 可选推送到邮箱或 Slack  

---

| 步骤 | 模块 | 功能 | 输出 | Emoji |
|------|------|------|------|-------|
| 1 | GitHub Actions | 每周定时触发 / 手动触发 | 无 | ⏱️ |
| 2 | Python Scraper | 抓取网站/API (STEM4kids, Art School, Robolabs, 图书馆活动) | 原始 JSON / CSV | 🔍 |
| 3 | 数据清洗 & 分类 | 去重、结构化、分类课程类型 (编程/艺术/舞蹈) | 整理好的 JSON | 🧹 |
| 4 | AI 智能处理 | 调用 GPT API 摘要课程信息 | Markdown 文件 + 推荐等级/标签 | 🤖 |
| 5 | 输出 & 存储 | Markdown 文件推送 GitHub Repo，JSON 文件存档，历史记录比对 | Markdown 文件 + JSON 文件 | 💾 |
| 6 | 可选通知 | 邮件 / Slack 推送课程摘要、链接、分类 | 邮件 / Slack 消息 | 📧 |

---

💡 **说明：**

- 每周自动运行，也可手动触发  
- 支持多网站抓取，可扩展课程分类  
- AI 处理生成 Markdown 文件，便于 GitHub 展示  
- 可选通知用于提醒新增或更新课程
